# 利润成本计算分析系统

全功能的财务利润成本分析工具，帮助企业管理收支、分析成本结构、获取智能优化建议。

## 功能特性

- **收支管理** — 添加/删除收入和支出条目，支持多类别分类
- **实时仪表盘** — 总收入、总支出、净利润、利润率一目了然
- **可视化图表** — 支出类别占比饼图、收入来源分析
- **智能建议** — 基于数据自动分析并给出降低成本、增加利润的具体建议
- **分析报告** — 一键生成完整财务分析报告，支持打印
- **本地存储** — 数据保存在浏览器中，刷新不丢失

## 使用方式

直接在浏览器打开 `index.html`，或：

```bash
npx serve .
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+I` | 聚焦收入输入框 |
| `Ctrl+E` | 聚焦支出输入框 |

## 技术栈

- HTML5 / CSS3 / 纯 JavaScript
- Canvas 图表绘制
- LocalStorage 数据持久化
- 响应式设计
