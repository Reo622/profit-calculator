# 💰 利润成本计算分析系统 Pro

全功能企业级财务分析工具，支持收支管理、盈亏平衡分析、预算规划、情景模拟、智能优化建议。

## 🚀 功能模块

### 📊 数据仪表盘
- 四大核心指标：总收入、总支出、净利润、利润率
- ECharts 交互式图表：月度收支趋势折线图、收支类别柱状图、来源占比饼图、利润率仪表盘
- 最近交易记录一览

### 💰 收支管理
- 收入/支出录入（支持 8 种支出类别 + 4 种收入类别）
- 批量导入（CSV 粘贴）
- CSV 文件导入/导出
- JSON 数据备份/恢复
- 收支明细表格查询

### 📈 盈亏平衡分析
- 计算边际贡献和边际贡献率
- 计算盈亏平衡点（数量和金额）
- 目标利润所需销量分析
- 安全边际评估
- ECharts 盈亏平衡可视化图表（收入线/总成本线/固定成本线）

### 📋 预算规划
- 按月设置各分类预算
- 预算执行率可视化（进度条）
- 预算 vs 实际对比柱状图
- 自动根据历史支出设置预算
- 差异分析和超支预警

### 🔮 情景模拟 (What-If)
- 5 个变量滑块：收入、原材料、人力、营销、其他
- 实时计算调整后的利润和利润率
- 利润变化幅度分析
- 情景保存和对比

### 🧠 智能建议
- 8 维度自动分析：盈利能力、成本集中度、营销ROI、人力成本、原材料成本、收入多元化、间接费用、税务优化
- 三级预警：严重🔴、警告🟡、良好🟢
- 每项建议包含具体金额影响和可操作方案

### 📊 完整报告
- 一键生成财务分析报告
- 支出结构分析
- TOP5 最大支出项
- 支持打印

### 核心算法说明

| 算法 | 公式 | 应用模块 |
|------|------|----------|
| 利润计算 | `净利润 = Σ收入 - Σ支出` | 仪表盘、报告 |
| 利润率 | `利润率 = 净利润 / 总收入 × 100%` | 仪表盘、建议 |
| 边际贡献 | `边际贡献 = 售价 - 单位变动成本` | 盈亏平衡 |
| 盈亏平衡点 | `BE = 固定成本 / 边际贡献` | 盈亏平衡 |
| 目标利润销量 | `目标销量 = (固定成本 + 目标利润) / 边际贡献` | 盈亏平衡 |
| 营销ROI | `ROI = 销售收入 / 营销支出` | 智能建议 |
| 安全边际 | `安全边际 = (实际收入 - BE收入) / 实际收入 × 100%` | 盈亏平衡 |
| 情景模拟 | `新利润 = 收入×(1+r%) - Σ(成本ᵢ×(1+rᵢ%))` | 情景模拟 |
| 预算执行率 | `执行率 = 实际支出 / 预算金额 × 100%` | 预算规划 |

## 📦 快速开始

### 方式一：直接打开
双击 `index.html` 在浏览器中运行（使用 LocalStorage 存储）

### 方式二：启动后端服务
```bash
npm install
npm start
```
然后访问 `http://localhost:3456`

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+1~6` | 切换标签页 |
| `Ctrl+I` | 聚焦收入输入 |
| `Ctrl+E` | 聚焦支出输入 |
